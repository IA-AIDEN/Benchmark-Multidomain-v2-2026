\[ v_f^2 = v_0^2 + 2g(h_{max} - h_0) \]

\[ h_{max} = h_0 + \frac{v_0^2}{2|g|} \]

\[ v_f = v_0 + gt \]

\[ 0 = 20 + (-9.8)t \]
\[ t = \frac{-20}{-9.8} \approx 2.04 \text{ segundos} \]

Entonces, el tiempo necesario para llegar al punto más alto es aproximadamente **2.04 segundos**.

#### Etapa 2: Descenso desde el punto más alto hasta el suelo

\[ t \approx \sqrt{6.21} \approx 2.49 \text{ segundos} \]

El tiempo de descenso es aproximadamente **2.49 segundos**.

Finalmente, sumamos ambos tiempos para obtener el tiempo total:

\[ T_{total} = t_{ascenso} + t_{descenso} \]

class DetectorAnomaliasStreaming:

def __init__(self, ventana=20):
self.buffer = []
self.medida_central = None # Inicializado más tarde dependiendo de la estrategia elegida
self.umbral_anomalia = 3 # Valor arbitrario inicial; debe ajustarse según contexto específico

self.ventana = ventana # Tamaño máximo del buffer

def agregar_dato(self, nuevo_valor):

if len(self.buffer) >= self.ventana:
self.buffer.pop(0)

self.buffer.append(nuevo_valor)

if not self.medida_central:
self.calcular_medida_central()

self.evaluar_nuevo_valores()

def calcular_medida_central(self):

if len(self.buffer) > 0:
self.medida_central = stats.median(self.buffer)
print(f"Nueva medida central establecida en {self.medida_central}")

def evaluar_nuevo_valores(self):

for val in self.buffer[-1:]:
