# Evaluation Scale

This scale was used to evaluate all benchmark responses under human qualitative assessment.

| Score | Meaning |
|---|---|
| 0 | Bad |
| 1 | Incorrect |
| 2 | Basic |
| 3 | Correct |
| 4 | Good |
| 5 | Excellent |

## Interpretation

- Scores reflect response quality, not just correctness
- A score of 5 requires both correctness and clarity
- A score of 3 indicates acceptable but not optimized performance
